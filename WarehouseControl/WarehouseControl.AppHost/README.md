# Warehouse control

Este proyecto esta desarrollado para cumplir con los siguientes requisitos:
1. Los productos llegarán al almacén en cajas 
2. Se almacenarán en Estanterías indicando su stock 
3. Los productos pueden estar en los siguientes estados: EN TRANSITO, ALMACENADO, ACABADO, EN TIENDA, CAMINO A TIENDA.
4. Las tiendas podrán solicitar más productos al almacén

## Tecnologías usadas
- Aspire Net
- Minimal APIs
- Service bus/RabbitMQ
- MongoDB
- Postgres
- Entityframework
- Blazor
